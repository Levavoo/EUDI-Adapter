# Scratch Notebook Templates

This folder contains reusable Markdown templates for the Scratch Notebook.

Use these templates to keep the notebook consistent without making it too formal.

---

## Template Types

| Template | Purpose |
|---|---|
| `YYYY-MM-DD_scratch_notes_template.md` | Daily scratch file structure |
| `YYYY-W##_scratch_review_template.md` | Weekly cleanup and movement review |
| `type_idea_template.md` | Capture possible features or concepts |
| `type_question_template.md` | Capture unclear topics and research questions |
| `type_remark_template.md` | Capture observations and reminders |
| `type_problem_template.md` | Capture confusion, blockers, and risks |
| `type_possible_task_template.md` | Capture possible tasks before they become real project work |
| `type_link_template.md` | Capture links and sources to check later |
| `type_decision_candidate_template.md` | Capture possible decisions before confirming them |
| `type_move_later_template.md` | Mark notes that should be moved into another notebook |

---

## Recommended Use

Use the daily template for normal work.

Use type templates only when a note deserves a little more structure.

Do not force every note into a detailed template. The Scratch Notebook should stay fast.
