### Issue 1: Root URL Returned `Not Found`

Result:

```json
{"detail":"Not Found"}
```

Cause:

No root endpoint `/` was defined in the FastAPI application.

Resolution:

Add a root route to `backend/app/main.py`.

### Issue 2: Python Code Was Pasted Into PowerShell

PowerShell showed parser errors after Python route code was pasted directly into the terminal.

Cause:

PowerShell does not understand Python syntax such as decorators:

```python
@app.get("/")
```

Resolution:

Add Python code inside `.py` files, then use PowerShell only to run commands.

### Issue 3: Local Folder Was Not a Git Repository

Initialize Git locally and connect the GitHub remote.


All protocol files must be stored in:

```text
docs/protocol/
```


### Writing Style

Use short, factual notes.

Record what was actually done.

Place unfinished work under `Open Tasks`.
